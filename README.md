# HTML5_Gaming
Shoot'em up game made in Javascript with CreateJs    
[Demo](https://oli8.github.io/HTML5_Gaming)  

## Controls
**Move:** arrow keys, (ZQSD) or (OKLM)  
**Shoot:** space bar or ctrl  
**Pause:** P or escape
